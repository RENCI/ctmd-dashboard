API: Available Endpoints

Proposals

  • Single Proposal
    Returns proposals in proposal table with given :id.
    api/proposals/:id
  • All Proposals
    Returns array of all proposals as object with the following attributes: including proposal_id (integer), pi_name (string), tic_name (string), proposal_status (integer), and submission_date (string, e.g., Wed Oct 26 2016).
    api/proposals
  • Proposal - By Stage
    Returns array of stages and array of proposals in each stage.
    api/proposals/by-stage
  • Proposal - By TIC/RIC
    Returns array of TICs and array of proposals in each stage.
    api/proposals/by-tic
  • Approved Proposals - Approved Services
    Returns array of approved proposals and services (given as ids) for which they are approved.
    api/proposals/approved-services
  • Submitted Proposals - New Service Selection
    Returns array of submitted proposals and services (given as ids).
    api/proposals/submitted-services

    • Services

    • Approval Services
      Returns services for proposals in the approval stage.
      api/services/approval
    • Submission Services
      Returns services for proposals in the submission stage.
      api/services/submission

    Stages

    • Stages
      Returns array of stages &mdash index (intenger), description (string) — in which proposals can be.
      api/stages

    Organizations

    • Submitting Organizations
      Returns array of organizations from which proposals were submitted as object &mdash index (intenger), description (string).
      api/organziations

    PIs

    • PIs
      Returns array of PIs in "PI" table.
      api/pis

    TICs

    • TICs
      Returns array of TIC & RIC names with their indices, as they appear in the name table.
      api/tics

    Therapeutic Areas

    • Therapeutic Areas
      Returns array of therapeutic areas indices, as they appear in the name table.
      api/tics

    Visualizations

    • Proposals Network
      Returns proposals with details required for network visualization, including proposal_id (integer), proposal_status (string), tic_name (string), org_name (integer), protocol_status (integer), anticipated_budget (string), funding_duration (string), and pi_name (string).
      api/services/approval
    • Submission Services
      Returns services for proposals in the submission stage.
      api/services/submission
